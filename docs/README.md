## 文档语言风格

- **代码、API、字段名**：英文
- **说明文案**：中文
- **标题**：中英混排（如 "API接口文档"）
- **术语表**：统一使用标准术语

## 贡献指南

添加新文档时请遵循以下规则：

1. 将文档放到正确的目录下
2. 使用中文标题，英文文件名
3. 在本文档中添加链接
4. 保持文档格式一致性
