# H5用户使用指南

本文档介绍用户在H5端使用问卷的流程。

## 访问问卷

### 方式一：通过链接访问

管理员分享的问卷链接，如：

```
https://your-domain.com/abc123
```

### 方式二：扫码访问

管理员提供的二维码，扫描后自动打开问卷页面

---

## 使用流程

```
输入密码 → 验证通过 → 查看介绍 → 开始答题 → 获取结果

```

### 1. 输入访问密码

在密码输入框中输入管理员提供的密码。

> 密码有效期为12小时，过期后需要联系管理员重新获取。

### 2. 验证密码

点击"开始测试"按钮验证密码。

- 密码正确：进入问卷介绍页
- 密码错误：提示"密码错误"
- 密码过期：提示"密码已过期"

### 3. 查看问卷介绍

通过密码验证后，显示问卷介绍页面：

- 问卷标题
- 问卷描述
- 点击"开始测试"按钮进入答题

### 4. 开始答题

**答题规则**：

- 单选题：只能选择一个选项
- 多选题：可以选择多个选项
- 上一题/下一题：可以切换题目
- 进度条：显示当前进度

**操作提示**：

- 点击选项即可选中
- 点击"下一题"进入下一题
- 点击"上一题"返回修改
- 必须完成所有题目才能提交

### 5. 查看测试结果

提交答案后，系统计算并显示结果：

- 结果类型：如 "ESTJ"、"推荐者"
- 结果标题
- 详细描述
- 匹配度分数（如果有）

---

## 常见问题

### Q1: 密码忘记了怎么办？

请联系问卷管理员获取新的访问密码。

### Q2: 答题中途可以暂停吗？

当前版本不支持暂停，建议一次性完成所有题目。

### Q3: 刷新页面会丢失进度吗？

会丢失，建议一次性完成。如需继续，需要重新输入密码验证。

### Q4: 可以重新测试吗？

可以，用同一密码重新访问问卷即可。

### Q5: 测试结果准吗？

问卷结果仅供参考，不同计分模式会有不同的结果呈现。

## 设备兼容

| 设备 | 支持情况 |
|-----|---------|
| 手机浏览器 | 完全支持 |
| 平板浏览器 | 完全支持 |
| 桌面浏览器 | 支持（自动缩放到移动端布局） |

---

## 界面说明

### 设计风格

- **暖米色调**：背景 `#f5f0eb`，文字 `#3a2e22`
- **字体**：衬线字体呈现优雅风格
- **响应式**：自动适配不同屏幕尺寸

### 页面结构

| 页面 | 功能 |
|-----|------|
| 密码验证页 | 输入密码验证身份 |
| 问卷介绍页 | 展示问卷标题和描述 |
| 答题页面 | 逐题展示题目和选项 |
| 结果页面 | 显示测试结果 |

---

## 隐私说明

- 系统不保存用户的详细答题记录
- 每次测试结果仅在本次访问中返回
- 不收集个人身份信息
